FAQ

Common questions.

Who is this product for? +
This pack is built for developers, freelancers, and technical users. You should be comfortable running Python scripts, editing code files, and basic server/hosting concepts. It is not designed for non-technical users.
How much does it cost to run the bot? +
The bot supports two AI providers. Google Gemini free tier: 1,500 requests/day, completely free, no credit card needed, never expires. Anthropic Claude: no permanent free tier, ~$5 trial credits for new accounts (expires), then ~$0.01 per conversation. We recommend starting with Gemini free tier.
Does the widget work on my website immediately? +
The widget works locally for testing right away. To use it on a live website, you need a small backend server (e.g. on Railway or Render) due to browser CORS restrictions. This is clearly documented in the included README. Deployment is your responsibility.
Can I use this for multiple businesses? +
Yes. You can use the code for as many of your own businesses as you want. Each one needs its own API key and company info setup.
What AI model does it use? +
It uses Claude Haiku by Anthropic — fast, affordable, and great for customer support. You can change the model in the code if you prefer a different one.
Is there a refund policy? +
Yes. If you purchase and the product doesn't work as described, contact us within 7 days for a full refund.
